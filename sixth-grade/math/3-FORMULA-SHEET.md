## NUMBER OPERATIONS

### Order of Operations (PEMDAS)
```
1. Parentheses    (  )
2. Exponents      x²
3. Multiply/Divide  x ÷  (left to right)
4. Add/Subtract   + -   (left to right)
```
**Example:** 2 + 3 x 4 = 2 + 12 = **14** (multiply before adding!)

| Rule | Formula | Example |

| Multiplying | x^a · x^b = x^(a+b) | x³ · x² = x⁵ |

## SCIENTIFIC NOTATION

### Large Numbers
```
a × 10^n    where 1 ≤ a < 10 and n is positive
```
**Example:** 3,200,000 = **3.2 × 10⁶**

### Small Numbers
```
a × 10^(−n)    where 1 ≤ a < 10 and n is positive
```
**Example:** 0.00047 = **4.7 × 10⁻⁴**

### Multiplying in Scientific Notation
```
(a × 10^m)(b × 10^n) = (a·b) × 10^(m+n)
```
**Example:** (3 × 10⁴)(2 × 10³) = 6 × 10⁷

### Dividing in Scientific Notation
```
(a × 10^m) ÷ (b × 10^n) = (a÷b) × 10^(m−n)
```
**Example:** (8 × 10⁶) ÷ (2 × 10²) = 4 × 10⁴

## SQUARE ROOTS

### Simplifying
```
√(a·b) = √a · √b
```
**Example:** √50 = √(25·2) = 5√2

### Multiplying
```
√a · √b = √(a·b)
```
**Example:** √3 · √12 = √36 = 6

### Rationalizing the Denominator
```
a/√b = (a·√b)/(b)
```
**Example:** 6/√3 = 6√3/3 = 2√3

### Perfect Squares (MEMORIZE THESE!)
```
1² = 1      5² = 25     9² = 81      13² = 169
2² = 4      6² = 36     10² = 100    14² = 196
3² = 9      7² = 49     11² = 121    15² = 225
4² = 16     8² = 64     12² = 144
```

## FRACTIONS

### Adding/Subtracting (different denominators)
```
a/b ± c/d = (ad ± bc) / bd
```
**Example:** 1/3 + 1/4 = (4 + 3)/12 = 7/12

### Multiplying
```
a/b × c/d = ac/bd
```
**Example:** 2/3 × 4/5 = 8/15

### Dividing (flip and multiply)
```
a/b ÷ c/d = a/b × d/c = ad/bc
```
**Example:** 2/3 ÷ 4/5 = 2/3 × 5/4 = 10/12 = 5/6

### Complex Fractions
```
(a/b) / (c/d) = a/b × d/c
```
**Example:** (3/4)/(5/6) = 3/4 × 6/5 = 18/20 = 9/10

## PERCENT

### Three Percent Equations
```
Part = Percent × Whole       →  "What is 25% of 80?"    0.25 × 80 = 20
Percent = Part ÷ Whole       →  "15 is what % of 60?"   15/60 = 0.25 = 25%
Whole = Part ÷ Percent       →  "20 is 40% of what?"    20/0.40 = 50
```

### Percent of Change
```
Percent Change = (Amount of Change / Original Amount) × 100
```
**Example:** Price goes from $40 to $50 → change = $10 → (10/40) × 100 = **25% increase**

### Converting Between Forms
```
Fraction → Decimal:    divide numerator by denominator     3/4 = 0.75
Decimal → Percent:     multiply by 100                     0.75 = 75%
Percent → Decimal:     divide by 100                       75% = 0.75
Percent → Fraction:    put over 100, simplify              75% = 75/100 = 3/4
```

## RATIOS & PROPORTIONS

### Proportion (Cross-Multiply to Solve)
```
a/b = c/d    →    a · d = b · c
```
**Example:** 3/4 = x/20 → 3 × 20 = 4x → 60 = 4x → x = 15

### Ratio Problems with Totals
```
Part = (ratio part / ratio total) × actual total
```
**Example:** Ratio 3:5, total = 40 → first group = (3/8) × 40 = 15

## ALGEBRA

### Solving Two-Step Equations
```
ax + b = c
Step 1: Subtract b from both sides    →  ax = c − b
Step 2: Divide both sides by a         →  x = (c − b)/a
```
**Example:** 3x + 5 = 20 → 3x = 15 → x = 5

### Distributive Property
```
a(b + c) = ab + ac
a(b − c) = ab − ac
```
**Example:** 4(x + 3) = 4x + 12

### FOIL (Multiplying Binomials)
```
(a + b)(c + d) = ac + ad + bc + bd
```
**Example:** (x + 3)(x + 2) = x² + 2x + 3x + 6 = x² + 5x + 6

### Slope of a Line
```
m = (y₂ − y₁) / (x₂ − x₁)    or    m = rise / run
```
**Example:** Points (1, 2) and (3, 8) → m = (8−2)/(3−1) = 6/2 = 3

### Slope-Intercept Form
```
y = mx + b
    m = slope
    b = y-intercept (where line crosses y-axis)
```
**Example:** y = 2x + 3 → slope is 2, crosses y-axis at (0, 3)

### Graphing Using Intercepts
```
x-intercept: set y = 0, solve for x
y-intercept: set x = 0, solve for y
```

### Inequality Rule
```
When you multiply or divide by a NEGATIVE number → FLIP the inequality sign
```
**Example:** −2x > 6 → x < −3

## SEQUENCES

### Arithmetic Sequence (add same number each time)
```
aₙ = a₁ + (n − 1)d
    a₁ = first term
    d  = common difference
    n  = term number
```
**Example:** 2, 5, 8, 11... → a₁ = 2, d = 3 → a₁₀ = 2 + 9(3) = 29

### Geometric Sequence (multiply same number each time)
```
aₙ = a₁ · r^(n−1)
    a₁ = first term
    r  = common ratio
```
**Example:** 3, 6, 12, 24... → a₁ = 3, r = 2 → a₅ = 3 · 2⁴ = 48

## VARIATION

### Direct Variation
```
y = kx        (k is a constant)
```
**Example:** If y = 12 when x = 4, then k = 3, so y = 3x

### Inverse Variation
```
y = k/x    or    xy = k
```
**Example:** If y = 6 when x = 2, then k = 12, so y = 12/x

## GEOMETRY — ANGLES

### Angle Facts
```
Complementary angles:   add up to 90°
Supplementary angles:   add up to 180°
Vertical angles:        are EQUAL
Triangle angles:        add up to 180°
Quadrilateral angles:   add up to 360°
```

### Interior Angles of a Polygon
```
Sum = (n − 2) × 180°     where n = number of sides
```
**Example:** Hexagon (6 sides): (6−2) × 180° = 720°

### Each Interior Angle of a REGULAR Polygon
```
Each angle = (n − 2) × 180° / n
```
**Example:** Regular hexagon: 720°/6 = 120°

## GEOMETRY — 2D SHAPES

### Rectangle
```
Perimeter = 2l + 2w
Area = l × w
```

### Square
```
Perimeter = 4s
Area = s²
```

### Triangle
```
Perimeter = a + b + c
Area = ½ × base × height
```

### Parallelogram
```
Area = base × height     (height is perpendicular!)
```

### Trapezoid
```
Area = ½(b₁ + b₂) × h     (b₁ and b₂ are the parallel sides)
```
**Example:** bases = 6 and 10, height = 4 → A = ½(16)(4) = 32

### Circle
```
Circumference = π × d = 2πr
Area = πr²
```
**Example:** r = 5 → C = 10π ≈ 31.42, A = 25π ≈ 78.54

### Arc Length (for central angle θ)
```
Arc Length = (θ/360) × 2πr
```

## GEOMETRY — 3D SHAPES

### Rectangular Prism (Box)
```
Volume = l × w × h
Surface Area = 2lw + 2lh + 2wh
```
**Example:** 3 × 4 × 5: V = 60, SA = 2(12) + 2(15) + 2(20) = 94

### Cube
```
Volume = s³
Surface Area = 6s²
```

### Cylinder
```
Volume = πr²h
Surface Area = 2πr² + 2πrh
```
**Example:** r = 3, h = 10: V = π(9)(10) = 90π ≈ 282.74

### Pyramid
```
Volume = ⅓ × Base Area × height
Surface Area = Base Area + ½ × Perimeter × slant height
```

### Cone
```
Volume = ⅓πr²h
Surface Area = πr² + πrl     (l = slant height)
```

### Sphere
```
Volume = (4/3)πr³
Surface Area = 4πr²
```
**Example:** r = 6: V = (4/3)π(216) = 288π, SA = 4π(36) = 144π

## PYTHAGOREAN THEOREM & TRIGONOMETRY

### Pythagorean Theorem (right triangles only!)
```
a² + b² = c²
    a, b = legs
    c = hypotenuse (longest side, opposite the right angle)
```
**Example:** a = 6, b = 8 → c² = 36 + 64 = 100 → c = 10

### Common Pythagorean Triples (MEMORIZE!)
```
3, 4, 5          and multiples like 6, 8, 10
5, 12, 13        and multiples like 10, 24, 26
8, 15, 17
7, 24, 25
```

### Special Right Triangles
```
45-45-90:    sides are   1 : 1 : √2
30-60-90:    sides are   1 : √3 : 2
```
**Example (45-45-90):** If each leg = 7, then hypotenuse = 7√2
**Example (30-60-90):** If short leg = 5, long leg = 5√3, hypotenuse = 10

### Trigonometric Ratios — SOH CAH TOA
```
sin(θ) = Opposite / Hypotenuse        (SOH)
cos(θ) = Adjacent / Hypotenuse        (CAH)
tan(θ) = Opposite / Adjacent          (TOA)
```
**Example:** Right triangle with angle θ, opposite = 3, adjacent = 4, hypotenuse = 5:
- sin(θ) = 3/5 = 0.6
- cos(θ) = 4/5 = 0.8
- tan(θ) = 3/4 = 0.75

## PROBABILITY

### Basic Probability
```
P(event) = favorable outcomes / total outcomes
```
**Always between 0 and 1 (or 0% and 100%)**

### Complement
```
P(not A) = 1 − P(A)
```

### Independent Events (Multiplication Rule)
```
P(A and B) = P(A) × P(B)
```
**Example:** Coin flip (heads) AND die roll (6): ½ × ⅙ = 1/12

### Dependent Events
```
P(A and B) = P(A) × P(B given A already happened)
```

### Geometric Probability
```
P = favorable area / total area
```

## SCALE & SIMILARITY

### Scale Factor Effects
```
If dimensions are multiplied by scale factor k:
  Perimeter  → multiplied by k
  Area       → multiplied by k²
  Volume     → multiplied by k³
```
**Example:** Scale factor k = 3 → Area is 9× bigger, Volume is 27× bigger

## INTEREST & GROWTH

### Simple Interest
```
I = P × r × t
    I = interest earned
    P = principal (starting amount)
    r = rate (as decimal)
    t = time (in years)
```
**Example:** $500 at 4% for 3 years: I = 500 × 0.04 × 3 = $60

### Compound Interest / Growth
```
A = P(1 + r)^t
```
**Example:** $1000 at 5% for 2 years: A = 1000(1.05)² = $1102.50

### Decay
```
A = P(1 − r)^t
```
**Example:** $800 loses 10%/year for 3 years: A = 800(0.9)³ = $583.20
