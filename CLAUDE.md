# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## What This Repo Is

A collection of markdown study materials for **Saxon Math Course 3** (6th grade, Lessons 1-120). There is no application code, build system, or test suite.

## Repository Structure

```
sixth-grade/math/
  1-STUDY-GUIDE.md    -- Comprehensive topic-by-topic guide (17 parts, all 120 lessons)
  2-PRACTICE-TEST.md  -- 50-question test with answer key and scoring guide
  3-FORMULA-SHEET.md  -- Quick-reference formula sheet organized by topic
```

Files are numbered to indicate the intended reading/usage order: study first, then practice, with the formula sheet as a reference companion.

## Content Conventions

- Materials are written in a casual, encouraging tone aimed at a 6th grader
- Mathematical notation uses plain text (e.g., `sqrt(2)`, `x^2`, `pi`) and Unicode symbols (e.g., `x²`, `π`) rather than LaTeX
- The study guide references specific Saxon Math lesson numbers (e.g., "Lesson 31") so topics can be traced back to the curriculum
- The practice test is designed for ~90 minutes, no calculator unless specified
