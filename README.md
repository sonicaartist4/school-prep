# School Prep

Study materials for school test preparation.

## Sixth Grade Math

Resources covering **Saxon Math Course 3** (Lessons 1-120). Use them in order:

1. **[Study Guide](sixth-grade/math/1-STUDY-GUIDE.md)** - Topic-by-topic walkthrough of all 120 lessons with examples
2. **[Practice Test](sixth-grade/math/2-PRACTICE-TEST.md)** - 50-question test (~90 min, no calculator) with answer key
3. **[Formula Sheet](sixth-grade/math/3-FORMULA-SHEET.md)** - Quick-reference formulas organized by topic
4. **[Tips and Tricks](sixth-grade/math/4-TIPS-AND-TRICKS.md)** - Shortcuts, memory tricks, and strategies for working faster

### Topics Covered

- Number sense, integers, fractions, decimals, percents
- Algebra (equations, inequalities, slope-intercept form, FOIL)
- Exponents, roots, scientific notation
- Ratios, proportions, direct & inverse variation
- Geometry (angles, polygons, area, volume, surface area)
- Pythagorean theorem & trigonometry (SOH-CAH-TOA)
- Probability, statistics, sequences
- Compound interest, growth & decay
- Unit conversions & measurement
